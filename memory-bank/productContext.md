# Product context

## Problem

Recruiters need a single place to register candidates with structured education and experience data and optional CV files, with validation and clear success or error feedback.

## User experience goals

- Obvious entry point from the recruiter dashboard
- Accessible form (labels, error announcements, keyboard-friendly controls)
- Responsive layout (Tailwind)
- Autocomplete hints for institution and company based on data already stored

## Personas

- **Recruiter**: creates candidates; uses dashboard → add candidate flow
